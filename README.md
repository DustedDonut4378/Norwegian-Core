# Norwegian-Core
An Anki deck for Norwegian (Bokmål)

This deck was largely generated by AI. I used a script to create example sentences from a cleaned Norwegian Kelly list.

Before sentence generation I cleaned the Kelly list by:
1. Removing English words.
2. Removing abbreviations.
3. Removing vulgar words.
4. Correcting parts of speech.
For items removed in steps 1–3 I substituted other useful Norwegian words not already in the list.

Sentence generation pipeline (using Gemini 3.1 Pro):
1. Generate sentences in batches of 10 for CEFR ranges: A1 (ranks 1–1000), A2 (1001–3000), B1 (3001–6000).
2. Send each batch to three separate LLM instances for analysis and correction.
3. Combine the original batch plus the three critiques and send to the model one final time to produce the finalized batch.
4. Save the final output to CSV and import into Anki.

TTS:
- Audio synthesized with Gemini 3.1 Flash TTS.
